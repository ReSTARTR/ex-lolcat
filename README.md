Lolcat
======

This is the clone of [busyloop/lolcat](https://github.com/busyloop/lolcat).
But, not supports animation and some features of the original.

## Compile

```bash
$ mix deps.get && mix escript.build
```

## Run

```bash
$ cowsay hello,world | ./lolcat
```
